# Stack Overflow Developer Survey 2024 Analysis 📊

## 🌟 Project Overview
This repository contains a comprehensive data analysis of the **2024 Stack Overflow Developer Survey**. The project explores the global developer landscape, focusing on professional experience, programming language popularity, and the economic factors of remote work.

The analysis processes a dataset of over **65,000 responses** to extract meaningful insights for the tech industry.

## 🔍 Key Research Questions
- What is the average professional experience level among global developers?
- How does **Python** adoption vary across different age demographics?
- Which countries offer the highest compensation for Python specialists?
- What industries dominate the high-paid **remote work** sector (top 25% earners)?

## 🛠️ Tech Stack & Methods
- **Language:** Python 3.10+
- **Library:** [Pandas](https://pandas.pydata.org/) for data manipulation and [NumPy](https://numpy.org/) for numerical operations.
- **Techniques:** - Data Cleaning & Handling Missing Values (NaN)
  - Descriptive Statistics (Mean, Median, Mode)
  - Grouping & Aggregation
  - Quantile-based Filtering (Q3 for high earners)
  - String Operations for Tech Stack Analysis

## 📈 Key Findings
- **The Python Effect:** Python usage peaks in the "under 25" age group, showing a strong trend toward the language in modern tech education.
- **The Remote Premium:** High-compensated remote roles are heavily concentrated in the **Financial Services** and **Healthcare** sectors.
- **Global Markets:** Beyond the US, countries like **Andorra** and **Isle of Man** show surprisingly high median compensations due to niche financial hubs.
